# Spring AI - ElevenLabs

This module provides Spring AI integration with ElevenLabs AI audio services:

- **Text-to-Speech (TTS)**: Convert text into natural-sounding speech with customizable voices
- **Speech-to-Text (STT)**: Transcribe audio files into text with support for multiple languages, speaker diarization, and word-level timestamps
